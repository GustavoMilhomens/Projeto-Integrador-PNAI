# Projeto PNAI
Estamos Criando um Site que mostra uma lista de presença para facilitar os ajudantes no PNAE
Estamos desenvolvendo um site com o objetivo de facilitar o controle da lista de presença do Programa Nacional de Alimentação Escolar (PNAE) no IFTO. A proposta é criar uma plataforma digital simples, organizada e de fácil utilização, voltada para auxiliar os servidores e colaboradores responsáveis pelo registro da presença dos alunos atendidos pelo programa.

Atualmente, o controle de presença pode demandar tempo e estar sujeito a falhas quando realizado manualmente. Com o site, será possível registrar a presença de forma rápida e segura, organizar as informações por turma, curso ou período e manter os dados armazenados de maneira estruturada. Isso contribui para reduzir erros, evitar retrabalho e melhorar a organização das informações.

Além disso, o sistema permitirá acompanhar os registros de forma mais eficiente, facilitando consultas, conferências e possíveis relatórios necessários para controle interno e prestação de contas. A interface será intuitiva, pensada para que qualquer responsável consiga utilizar a ferramenta sem necessidade de conhecimento técnico avançado.

O principal objetivo do projeto é tornar o processo de controle de presença mais prático, confiável e eficiente, colaborando para a melhoria da gestão do PNAE no IFTO e oferecendo mais agilidade no atendimento às demandas administrativas.
